# Собрать и запусить Docker

1. создаем директорию и переходим:
```bash
mkdir test-project
```
2. создаем html и Docker file: 
```bash
echo "<h1> Hello from Ivan Tsvetkov! </h1>" >> index.html
```
 создаем Docker file:
```bash
cat > Dockerfile << 'EOF'
FROM nginx:alpine
COPY index.html /usr/share/nginx/html/
EOF
```
3. Создаем образ:
```
docker built -t my-web-app:latest .
```
4. Запускаем контейнер с пробросом портов:
```bash
docker run -d -p 8080:80 --name test-web-container my-web-app:latest 
```
5. проверяем статус ответа и содержимое страницы по адресу
```bash
curl -w "=== HTTP Status: %{http_code} ===\n" http://localhost:8080
```
6. выполняем команду 
```bash
docker-compose up
```
7. Для этого можно использовать монтирование volumes в docker-compose.
