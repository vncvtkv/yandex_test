# Команды Git для работы с изменениями и коммитами

## 1. Сохранить незакоммиченные изменения и переключиться на main

```bash
# Сохраняем текущие изменения в stash (хранилище)
git stash save "WIP: изменения для feature/junior-task"

# Переключаемся на ветку main
git checkout main
```

## 2. После исправления бага вернуться к своей работе

```bash

# Вернуться в свою ветку
git checkout feature/junior-task

# Восстановить сохраненные изменения из stash
git stash pop

# Или если хотите просто применить изменения, но оставить их в stash
git stash apply
```

## 3. Изменить сообщение последнего коммита (если НЕ опубликован в удаленный репозиторий)

```bash
# Изменить сообщение последнего коммита
git commit --amend -m "Новое сообщение коммита"
```
